# 格式化輸出方法 (printf)

```
System.out.printf("列印文字與格式", 變數_1, 變數_2, …);
```

左右括號內引數包含兩種資料，一者由兩個雙引號包起來 `("… ")` 的是文字直接輸出，以及列印變數的資料型態與指定格式；另一者為一連串所欲輸出的變數。

變數的列印格式由 `%` 作為標頭記號，每一個 `%` 表示一個變數內容；也就是說，有幾個變數輸出，就需要同樣多個 `%` 表示每一變數的輸出型態，較常用的輸出型態如以下表格所示。

| 列印格式  | 輸出敘述       | 列印格式 | 輸出敘述    |
| ----- | ---------- | ---- | ------- |
| %c    | 字元         | %o   | 8 進位整數  |
| %d    | 十進位整數      | %x   | 16 進位整數 |
| %5d   | 5 個位置整數    | \n   | 跳行      |
| %s    | 字串         | \t   | 跳 Tab 格 |
| %f    | 浮點數列印      | \’   | 印出單引號   |
| %5.2f | 5 位數及2 小數點 | \\"  | 印出雙引號   |
| %.2f  | 浮點數及2小數點   | \\\\ | 印出反斜線   |
